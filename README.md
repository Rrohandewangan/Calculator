# Calculator

A simple web-based calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations
- Clean and simple UI
- Instant result display

## Project Files
- `index.html`
- `style.css`
- `app.js`

## How to Run
1. Open `index.html` in your browser.
2. Use the buttons to perform calculations.
